## 프로젝트 소개
- 2개의 양수를 입력받고, 사칙연산 기호에 따라 연산을 수행하는 기본적인 계산기툴을 만들어보자
- java 문법이 익숙해지도록 다시 한번 더 실습을 해보자 ( NEW !! )

## 프로젝트 목표

### ◆ 필수 기능
#### Step 1. 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기

✅ 0 을 포함한 양의 정수 입력 : 사용자가 입력한 숫자 활용

✅ 사칙연산 기호 입력받기 : +, -, * , /

✅ 연산 수행 후, 결과값 출력하기 : if문 또는 switch문 활용

✅ 반복문 사용 하기 : "Exit" 입력 전까지 무한루프 생성

※ 예외문 : 
* 음수, 타 문자 입력 : "0을 포함한 양수만 입력가능합니다."
* 나눗셈에서 두번째 숫자를 0으로 입력 : "나눗셈 연산에서 분모 (두번째 정수)에 0이 입력될 수 없습니다."
* 설정된 연산자 외 (문자, 숫자 포함) : "지원하지 않는 연산자입니다"

#### Step 2. 클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기

✅ 결과값 반환 메서드 & 연산 결과 저장 필드 생성 : Calculator 클래스 생성

✅ main 메서드에서 Calculator 클래스 활용 : Step 1 의 App 클래스 사용

✅ 캡슐화 하기 : Getter, Setter 메서드 구현

✅ 가장 먼저 저장된 데이터 삭제하기

---

### ◆ 도전 기능
#### Step 3. Enum, 제네릭, 람다 & 스트림을 이해한 계산기 만들기

✅ Enum 상수를 활용해 정보 관리 하기 : ArithmeticCalculator 클래스 활용

✅ double 타입 값도 연산 가능하게 하기 : 제네릭 활용

✅ 저장된 연산 결과들 중 입력받은 값보다 큰 결과값 출력 : 람다, 스트림 활용

※ 예외조건 :
* ArithmeticCalculator (사칙연산 계산기): 양의 정수를 매개변수로 받아 연산
